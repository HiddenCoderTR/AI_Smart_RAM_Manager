Bu proje Windows™ [görev yöneticisine](https://learn.microsoft.com/tr-tr/shows/inside/task-manager) benzer şekilde çalışan, özellikle oyuncular ve genel kitle olmak üzere tüm kullanıcıları kapsayan bir performans yöneticisidir.
